# Agentic AI Financial Assistant

## Project Overview

This project is an Agentic AI-powered Personal Finance Assistant developed using Python and Streamlit.

The application analyzes personal finance data and provides interactive insights through charts and AI-driven responses.

---

## Features

- Total Expense Dashboard
- Average Expense
- Transaction Count
- Category-wise Spending
- Monthly Spending Trend
- Payment Method Distribution
- AI-powered Financial Insights
- Savings Recommendations

---

## Technologies Used

- Python
- Pandas
- Streamlit
- Plotly
- VS Code

---

## Project Structure

dashboard.py - Streamlit Dashboard

agentic_ai.py - AI Logic

read_data.py - Dataset Loader

personal_finance_dataset_8000_extended.csv - Dataset
